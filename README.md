# My Search-Enabled Chatbot

## Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
## Usage
CLI mode: python bot.py --cli

Web mode: python bot.py --web then visit http://localhost:5000
